1. Mengenal HTML, CSS, JavaScript & Tailwind

HTML (HyperText Markup Language) bahasa dasar untuk membuat struktur halaman web. Ia berfungsi seperti kerangka atau fondasi dari sebuah rumah,tanpa HTML halaman web tidak memiliki bentuk atau isi.

CSS (Cascading Style Sheets) bahasa yang digunakan untuk mengatur tampilan (desain) dari halaman web yang dibuat dengan HTML. Kalau HTML itu kerangkanya, maka CSS adalah cat, warna, dan gaya dekorasinya.

JavaScript adalah bahasa pemrograman utama di web yang membuat halaman menjadi interaktif dan dinamis. Kalau HTML itu kerangkanya dan CSS itu tampilannya, maka JavaScript adalah otaknya atau penggeraknya.

Tailwind CSS adalah framework CSS modern yang mempermudah pembuatan tampilan web dengan kelas utilitas siap pakai. Daripada menulis CSS panjang lebar, Tailwind memungkinkan kita menulis kelas langsung di HTML untuk mendesain elemen dengan cepat.

2. Tata Cara Membuat Website Sederhana

  1. Menentukan Ide dan Tujuan dari Website yang akan kita rancang.
  2. Siapkan Text editor seperti VS Code,untuk menampilakan code pemrograman.
  3. Buat sebuah folder lalu didalam folder tersebut buat tiga file yaitu: index.html, style.css, dan script.js.
  4. Isi struktur dasar HTML (dengan tambahkan elemen seperti header, paragraf, tombol, dan form sederhana).
  5. Styling dengan CSS (untuk mendesain warna, font, dan layout).
  6. Interaksi dengan JavaScript (untuk memberikan fungsi interaktif serta beberapa animasi).
  7. Menjalankan dan menguji Website (Buka index.html di browser klik dua kali atau buka lewat Live Server di editor).
  8. Publikasikan ke Web (Unggah semua file-mu ke layanan hosting (seperti vercel) agar website bisa diakses oleh semua orang.)
